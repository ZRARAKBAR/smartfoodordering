# Smart Ordering System

A Java Swing-based Smart Ordering System for restaurants, integrated with MySQL.  
This system allows customers to browse the menu, place orders, and track order status, while admins can manage the menu and view analytics.

## Features

- *Customer Interface*
  - Browse menu with item images
  - Add items to cart and place orders
  - View order status

- *Admin Panel*
  - Add, update, and delete menu items
  - View analytics: total orders, revenue, top-selling items
  - Manage menu categories

- *Kitchen Panel*
  - View and update pending orders
  - Track order preparation

## Technologies Used

- Java Swing (GUI)
- MySQL (Database)
- Object-Oriented Programming (OOP)
- MVC design pattern for code organization
